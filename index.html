Responsive - Mobile (articles & headers)

This code makes any articles and headers on the page responsive for viewing on mobile devices.

                    
@media screen and (max-width: 640px) {
    article {
        display: flex;
        flex-direction: column;
        flex-wrap: wrap;
        width: 100%;
        justify-content: center;
        height: 100%;
    }
    header {
        width: 100%;
    }
}

Make an Element a Flexbox

Turns this element into a flexbox

                    
element {
    display: flex;
    flex-direction: row;
    /* Centers along the main axis determined by flex-direction */
    justify-content: center;
    /* Centers along the opposite axis */
    align-items: center;
    width: 90%;
    /* Uses the height of the viewport */
    height: 100vh;
    margin: 0 auto;
    border-style: solid;
    border-width: 2px;
    }

Box Sizing

Allows for an Elements Padding and Border to be included in total Width and Height

                    
* {
    box-sizing: border-box;
    }

Pseudo-Classes

The Basic CSS Psuedo Classes can be used to give Elements different properties based on user action

                    
/* The user's cursor is positioned over the element */
button:hover {
    background-color: #772014;
    color: #fff;
}

/* The user is actively pressing down on the element */
button:active {
    font-size: 180%;
    box-shadow: 0 0 10px #000 inset;
}

/* The element is currently in focus, 
either by tabbing to or clicking on it */
button:focus {
    outline: none;
    border-color: #8ac4ff;
}

Style Selection Box

Allows for customization of Selection Boxes

                    
select {
    -moz-appearance: none;
    -webkit-appearance: none;
    appearance: none;
    background-image: url(../images/arrow.png);
    background-repeat: no-repeat;
    background-position: right;
    background-size: contain;
    }

CSS Variable

Allows for the creation of customizable CSS Variables

                    
:root {
    /* declare a variable with `--` syntax */
    --dark: darkred;
    --light: #fff;
    --navlink-color: lightpink;
    }
